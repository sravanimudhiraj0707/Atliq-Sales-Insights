# Atliq-Sales-Insights
# 📌 Project Overview
The Atliq Sales Insights Project is a data analysis project aimed at extracting meaningful insights from sales data using SQL and visualizing them through Power BI dashboards.
This project simulates a real-world business scenario where a company operates across multiple markets and needs data-driven decision-making to improve performance.
# 🎯 Objectives
Analyze overall sales performance
Identify top customers, products, and markets
Track revenue trends over time
Provide actionable insights for business growth
# 🗂️ Dataset Description
The dataset consists of multiple relational tables:
Customers – Customer details
Products – Product information
Transactions – Sales data (sales amount, quantity, date)
Markets – Region-wise market data
Date – Time-based attributes
# 🛠️ Tech Stack
SQL (MySQL) – Data extraction & analysis
Power BI – Data visualization
Excel – Data cleaning & validation
